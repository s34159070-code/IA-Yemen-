# هاشم الدرويش - تطبيق التواصل الاجتماعي الشامل

تطبيق تواصل اجتماعي متطور ومتكامل يجمع بين جميع الميزات الحديثة والمتقدمة. مبني باستخدام React Native و Expo مع دعم كامل للغة العربية وتصميم عصري احترافي.

**المطور:** هاشم عبدالرحمن عبدالحميد الدرويش الحريري

## الميزات الرئيسية

### 🏠 الشاشة الرئيسية
- عرض إحصائيات المستخدم (المتابعون، المتابعة، المنشورات)
- أزرار الإجراءات السريعة
- قائمة التريندات الحالية

### 📰 التغذية الاجتماعية (Feed)
- عرض المنشورات من المستخدمين
- نظام الإعجاب والتعليقات
- إمكانية المشاركة
- معلومات المؤلف والوقت

### 💬 الرسائل الفورية (Messages)
- قائمة المحادثات النشطة
- البحث عن المحادثات
- عداد الرسائل غير المقروءة
- إنشاء محادثات جديدة

### 📡 البث المباشر (Live Streaming)
- عرض البثوث المباشرة
- فلترة البثوث حسب الفئات
- عداد المشاهدين
- زر ابدأ بث مباشر

### 🛍️ المتجر (Marketplace)
- عرض المنتجات
- البحث والتصفية حسب الفئات
- نظام التقييمات
- عرض المراجعات
- سلة التسوق

### 🎮 الألعاب والتحديات (Games & Challenges)
- ألعاب تفاعلية متنوعة
- تحديات مثيرة
- نظام النقاط والمكافآت
- لوحة الصدارة

### ✨ الاكتشاف والتوصيات الذكية (AI Discover)
- توصيات ذكية مخصصة
- عرض سبب التوصية
- تحليل الاهتمامات

### 👤 الملف الشخصي (Profile)
- معلومات المستخدم الشاملة
- الإحصائيات الشخصية
- المنشورات السابقة
- الإعدادات والتفضيلات

## المتطلبات

- Node.js 22.13.0 أو أحدث
- pnpm 9.12.0 أو أحدث
- Expo CLI

## التثبيت والتشغيل

```bash
# استنساخ المستودع
git clone https://github.com/s34159070-code/IA-Yemen-.git
cd IA-Yemen-

# تثبيت المكتبات
pnpm install

# تشغيل التطبيق
pnpm dev
```

## الأوامر المتاحة

```bash
pnpm dev          # تشغيل التطبيق
pnpm build        # بناء للإنتاج
pnpm start        # بدء الخادم
pnpm check        # فحص الأخطاء
pnpm format       # تنسيق الكود
pnpm test         # تشغيل الاختبارات
pnpm android      # فتح على Android
pnpm ios          # فتح على iOS
```
