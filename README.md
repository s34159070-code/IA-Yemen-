# IA Yemen - تطبيق التواصل الاجتماعي اليمني الشامل

تطبيق تواصل اجتماعي يمني متطور ومتكامل يجمع بين جميع الميزات الحديثة والمتقدمة. مبني باستخدام React Native و Expo مع دعم كامل للغة العربية وتصميم عصري احترافي.

## الميزات الرئيسية

### 🏠 الشاشة الرئيسية
- عرض إحصائيات المستخدم (المتابعون، المتابعة، المنشورات)
- أزرار الإجراءات السريعة
- قائمة التريندات الحالية

### 📰 التغذية الاجتماعية (Feed)
- عرض المنشورات من المستخدمين
- نظام الإعجاب والتعليقات
- إمكانية المشاركة
- معلومات المؤلف والوقت

### 💬 الرسائل الفورية (Messages)
- قائمة المحادثات النشطة
- البحث عن المحادثات
- عداد الرسائل غير المقروءة
- إنشاء محادثات جديدة

### 📡 البث المباشر (Live Streaming)
- عرض البثوث المباشرة
- فلترة البثوث حسب الفئات (تقنية، موسيقى، طبخ، تصوير، رياضة، تعليم)
- عداد المشاهدين
- زر ابدأ بث مباشر

### 🛍️ المتجر (Marketplace)
- عرض المنتجات اليمنية
- البحث والتصفية حسب الفئات
- نظام التقييمات (النجوم)
- عرض المراجعات
- سلة التسوق

### 🎮 الألعاب والتحديات (Games & Challenges)
- **الألعاب:**
  - كويز الثقافة اليمنية
  - تحدي الكلمات
  - لعبة الذاكرة
  - تحدي الرياضيات
- **التحديات:**
  - تحدي الرقص اليمني
  - تحدي الطبخ
  - تحدي الفن والرسم
- نظام النقاط والمكافآت
- لوحة الصدارة

### ✨ الاكتشاف والتوصيات الذكية (AI Discover)
- توصيات ذكية مخصصة
- عرض سبب التوصية
- تحليل الاهتمامات
- توصيات الكتب والمنتجات والأصدقاء

### 👤 الملف الشخصي (Profile)
- معلومات المستخدم الشاملة
- الإحصائيات الشخصية
- المنشورات السابقة
- الإعدادات والتفضيلات

## المتطلبات

- Node.js 22.13.0 أو أحدث
- pnpm 9.12.0 أو أحدث
- Expo CLI

## التثبيت والتشغيل

```bash

git clone https://github.com/s34159070-code/IA-Yemen-.git
cd IA-Yemen-

# تثبيت المكتبات
pnpm install

# تشغيل التطبيق
pnpm dev
```

## الأوامر المتاحة

```bash
# تشغيل التطبيق في بيئة التطوير
pnpm dev

# تشغيل خادم التطوير فقط
pnpm dev:server

# تشغيل Metro Bundler فقط
pnpm dev:metro

# بناء التطبيق للإنتاج
pnpm build

# بدء خادم الإنتاج
pnpm start

# فحص الأخطاء
pnpm check

# تنسيق الكود
pnpm format

# تشغيل الاختبارات
pnpm test

# إنشاء قاعدة البيانات
pnpm db:push

# فتح التطبيق على Android
pnpm android

# فتح التطبيق على iOS
pnpm ios

# إنشاء رمز QR
pnpm qr
```
