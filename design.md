# Friendly Robot - تصميم الواجهة

## نظرة عامة
تطبيق Friendly Robot هو تطبيق تواصل اجتماعي يمني متكامل يوفر منصة للتواصل والتفاعل بين المستخدمين.

## قائمة الشاشات

### 1. الشاشة الرئيسية (Home Screen)
تعرض إحصائيات المستخدم والإجراءات السريعة والتريندات الحالية.

### 2. شاشة التغذية (Feed Screen)
تعرض منشورات المستخدمين مع إمكانية الإعجاب والتعليق والمشاركة.

### 3. شاشة الرسائل (Messages Screen)
قائمة بالمحادثات النشطة مع إمكانية البحث والرسائل الجديدة.

### 4. شاشة الملف الشخصي (Profile Screen)
معلومات المستخدم والإحصائيات والإعدادات.

### 5. شاشة تسجيل الدخول (Login Screen)
تسجيل الدخول والتسجيل الجديد.

### 6. شاشة البحث (Search Screen)
البحث عن المستخدمين والمنشورات والهاشتاجات.

## المحتوى الأساسي والوظائف

### الشاشة الرئيسية
- عرض إحصائيات المستخدم (المتابعون، المتابعة، المنشورات)
- أزرار الإجراءات السريعة (منشور جديد، اكتشف)
- قائمة التريندات الحالية

### شاشة التغذية
- عرض المنشورات مع معلومات المؤلف
- نظام الإعجاب والتعليقات
- إمكانية المشاركة
- وقت النشر

### شاشة الرسائل
- قائمة المحادثات النشطة
- آخر رسالة في كل محادثة
- عدد الرسائل غير المقروءة
- البحث عن المحادثات

## تدفقات المستخدم الرئيسية

### تدفق النشر
1. المستخدم يضغط على "منشور جديد"
2. يكتب محتوى المنشور
3. يضيف صور أو فيديو (اختياري)
4. ينشر المنشور
5. يظهر المنشور في التغذية

### تدفق الرسائل
1. المستخدم يذهب لشاشة الرسائل
2. يختار محادثة أو ينشئ محادثة جديدة
3. يكتب الرسالة
4. يرسل الرسالة
5. تظهر الرسالة في المحادثة

### تدفق المتابعة
1. المستخدم يبحث عن مستخدم آخر
2. يضغط على زر المتابعة
3. يصبح متابعاً للمستخدم
4. يرى منشورات المستخدم في التغذية

## معايير التصميم
- اتباع معايير Apple Human Interface Guidelines (HIG)
- تصميم موجه للهاتف بصيغة عمودية (9:16)
- دعم الاستخدام بيد واحدة
- تجربة مستخدم سلسة وسهلة
- دعم الوضع الفاتح والداكن
- استخدام الرموز التعبيرية للتفاعل
